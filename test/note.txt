post文件夹中为用户操作时向服务器发送修改请求的数据格式

login.json	用户登录
{
	"studentNo": "201330651123",
	"password": "xietao661919",
}
--------------------
logout.json	用户登出
{
	"acount": {
  	  "userid": "12",
  	  "token": "123442efc23"
  }
}
--------------------

{

      "userid": "12",
      "token": "123442efc23"
    },
  "content": {

    }
